## 👨‍💻 Sobre Mim

Sou um **Desenvolvedor Web Full Stack Júnior** e estudante de Sistemas de Informação, com experiência prática na criação de sistemas úteis e automação de processos corporativos. Busco sempre criar soluções que eliminem o retrabalho manual e tragam eficiência.

- 🔭 **Experiência:** Desenvolvimento full-stack, criação de APIs RESTful, dashboards de análise de dados e suporte técnico em ambientes de alta demanda.
- 🌱 **Aprendendo atualmente:** Node.js, Tailwind CSS, Zustand e Scikit-learn.
- 🎯 **Objetivo:** Construir sistemas eficientes e automatizar processos que resolvam problemas reais de negócios.
